ANIMATIONS & MICRO-INTERACTIONS
════════════════════════════════════════════════════════════

ANIMATIONS & MICRO-INTERACTIONS — Ce qui rend l'expérience mémorable

Philosophie : chaque animation a un sens. Jamais de mouvement gratuit.
Durées : micro 150ms / standard 250ms / narratif 400ms / transition 600ms

─── HERO ───
- Titre : fade-in + slide-up 40px (600ms, délai 200ms)
- Badge stat orange : scale 0→1 + bounce léger (500ms, délai 800ms)
- CTA : pulse glow teal toutes les 3s (invite passive à cliquer)
- Compteurs : count-up animé (0 → valeur en 1.5s, ease-out)
- Rotation des phrases d'accroche : crossfade toutes les 3s

─── SCROLL STORYTELLING ───
- Chaque section : IntersectionObserver → fade + translateY(-30px → 0)
- Barre de progression : largeur calculée en temps réel depuis scrollY
- Chiffres et stats : déclenchés uniquement quand visibles dans le viewport

─── CARDS THÉMATIQUES ───
- Hover : scale 1.02 + overlay teal + slide-up du sous-titre
- Click : ripple effect depuis le point de clic (cercle teal s'étend)
- Chargement initial : stagger d'apparition (carte 1 → 2 → 3, délai 80ms entre)

─── FILTRES AIDES ───
- Pill select : background fill depuis le centre (clip-path circle expand)
- Cartes filtrées : opacity 40% + scale 0.98 (250ms smooth)
- Compteur "X aides" : count-up à chaque changement de filtre

─── SIMULATEUR DE BUDGET ───
- Sliders : thumb teal, track remplie en teal
- Résultat : mise à jour temps réel, chiffre count-up à chaque drag
- Si solde négatif : résultat en orange + message d'alerte douce

─── TRANSITIONS ENTRE SECTIONS ───
- Flèche "→ Section suivante" : hover = slide vers la droite 6px
- Scroll to section : smooth scroll natif (scroll-behavior: smooth)

─── FEEDBACK GÉNÉRAL ───
- Copie d'un lien : toast "✓ Lien copié !" (coin bas-droit, 2.5s)
- Lien externe : icône ↗ apparaît au hover, ouvre dans un nouvel onglet
- Accordion ouvert : chevron rotate 180° (200ms)
- prefers-reduced-motion : toutes les animations désactivées si demandé
