ANIMATIONS & MICRO-INTERACTIONS
════════════════════════════════════════════════════════════

ANIMATIONS & MICRO-INTERACTIONS — Ce qui rend l'expérience mémorable

Philosophie : chaque animation a un sens. Jamais de mouvement gratuit.
Durées : micro 150ms / standard 250ms / narratif 400ms / transition page 600ms

─── HERO ───
- Titre : fade-in + slide-up 40px (600ms, délai 200ms)
- Badge stat : scale 0 → 1 + bounce (500ms, délai 800ms)
- CTA : pulse glow teal toutes les 3s (attention passive)
- Compteurs : count-up animé (0 → valeur en 1.5s, ease-out)
- Rotation des phrases d'accroche : crossfade toutes les 3s

─── SCROLL STORYTELLING ───
- Chaque section : IntersectionObserver → fade + translateY(-30px → 0)
- Barre de progression : largeur calculée depuis scrollY
- Chiffres/stats : déclenchés quand visibles dans viewport

─── CARDS THÉMATIQUES ───
- Hover : scale 1.02 + overlay teal + slide-up du sous-titre
- Click : ripple effect depuis le point de clic
- Badge "Exploré" : checkmark qui se dessine (stroke-dashoffset)

─── FORMULAIRES (connexion/inscription) ───
- Champ focus : border-color → teal + glow 0 0 0 3px teal/20%
- Validation : icône ✓ qui apparaît en scale depuis 0 (200ms)
- Erreur : shake horizontal 3 fois + border rouge
- Bouton submit loading : spinner + texte "Connexion en cours..."
- Stepper : étape complète → pastille verte avec checkmark animé

─── FILTRES AIDES ───
- Pill select : background fill depuis le centre (clip-path circle)
- Cartes filtrées : opacity 40% + scale 0.98 (250ms)
- Compteur "X aides" : count-up à chaque changement de filtre

─── FEEDBACK UTILISATEUR ───
- Toast notification (coin bas-droit) : slide-in 300ms, disparaît après 3s
  Exemples : "✓ Aide sauvegardée" / "✓ Profil mis à jour"
- Confirmation d'inscription : animation confetti légère (canvas, 2s)

MOBILE FIRST — Expérience tactile premium

Breakpoints :
- Mobile  : 320px–767px (prioritaire)
- Tablette : 768px–1023px
- Desktop : 1024px+

─── HERO MOBILE ───
- Titre H1 : 40px (pas 72px — lisible sans zoom)
- Stats et compteurs en ligne horizontale scrollable
- CTA pleine largeur, height 56px (zone tactile confortable)
- Phrase rotation : désactivée sur mobile → affiche la 1re seulement

─── NAVIGATION MOBILE ───
- Hamburger en haut à droite (44×44px min)
- Drawer latéral full-height depuis la droite
- Liens : 48px height, séparateurs subtils
- Barre de progression : fine ligne sous la navbar (2px)

─── CARDS THÉMATIQUES MOBILE ───
- 1 colonne, swipe horizontal possible (snap scroll)
- Indicateurs de position (dots) sous le carrousel
- Tap = navigation immédiate (pas de hover)
- Badge "🔒" visible sans hover

─── FORMULAIRES MOBILE ───
- Champs height 52px (doigt confortable)
- Keyboard type adapté : email/tel/password
- Stepper simplifié : "1 — Identité" / "2 — Profil" (texte court)
- Bouton CTA sticky en bas lors du scroll dans le formulaire

─── AIDES MOBILE ───
- Filtres : scroll horizontal (overflow-x auto, no scrollbar visible)
- Cards : une par ligne, accordion expand complet
- Liens "Accéder au site" : bouton pleine largeur

─── PERFORMANCE ───
- Images : WebP + lazy loading (loading="lazy")
- Fonts : subset latin + display=swap
- Animations : prefers-reduced-motion → désactivées si besoin
- Score Lighthouse cible : >90 perf / >95 accessibilité
